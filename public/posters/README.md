# Event Posters

Add your event poster images to this folder.

**Supported formats**: JPG, PNG, WebP
**Recommended size**: 1200x800px or 3:2 aspect ratio

## Naming Convention

Use descriptive names that match your event title:
- `ai-workshop.jpg`
- `hackathon-2026.png`
- `cloud-bootcamp.jpg`

After adding images, update the poster path in `src/data/events.json`:

```json
{
  "poster": "/posters/your-image-name.jpg"
}
```
